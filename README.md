# Панель аналитики товаров
Проект предоставляет инструменты для анализа данных, выявления аномалий, прогнозирования и визуализации финансовой или бизнес-статистики.

Структура проекта состоит из двух основных частей: **Frontend** (React) и **Backend** (FastAPI).

---

## 🚀 Структура проекта

* `frontend/` — Пользовательский интерфейс (React + Vite). Киберпанк-дашборд для визуализации данных.
* `backend/` — Серверная часть (FastAPI). Предоставляет API, интегрирует машинное обучение (`app/ml/`), работает с базой данных и вычисляет статистики.
* `scripts/` — Утилиты для загрузки и первичной обработки датасетов (например, импорт `.csv` в базу данных).
* `models/` — Локальное хранилище весов моделей машинного обучения или других артефактов.
* `notebooks` / `analysis` — (Если применимо) Jupyter-ноутбуки с EDA (Exploratory Data Analysis).

---

## 🛠 Требования для запуска

Перед началом убедитесь, что у вас установлены:
* **Python** (3.9+)
* **Node.js** (18+)
* **npm** (предустановлен с Node.js)
* **PostgreSQL** (или другая БД, если вы настроили `.env` в backend, уточните настройки базы данных).

---

## ⚙️ Локальный запуск (Инструкция)

### 1. Запуск Backend (FastAPI)

1. Откройте терминал и перейдите в папку бэкенда:
   ```bash
   cd backend
   ```
2. Создайте и активируйте виртуальное окружение:
   ```bash
   python -m venv venv
   # Для Windows:
   venv\Scripts\activate
   # Для MacOS/Linux:
   # source venv/bin/activate
   ```
3. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
4. Настройте переменные окружения:
   * Создайте файл `.env` в папке `backend/` (если нужно, используйте `.env.example` как шаблон).
   * Укажите параметры подключения к БД (например, строку `DATABASE_URL`).
5. (Опционально) Загрузите данные в базу:
   Если база пустая, вы можете использовать скрипты импорта:
   ```bash
   cd ../scripts
   python import_data_fast.py
   ```
6. Запустите сервер:
   ```bash
   uvicorn app.main:app --reload
   ```
   * *Backend будет доступен по адресу:* `http://localhost:8000`
   * *Документация API (Swagger):* `http://localhost:8000/docs`

### 2. Запуск Frontend (React)

1. Откройте **новый** терминал и перейдите в папку фронтенда:
   ```bash
   cd frontend
   ```
2. Установите зависимости Node.js:
   ```bash
   npm install
   ```
3. Запустите сервер для разработки:
   ```bash
   npm run dev
   ```
   * *Frontend (Дашборд) будет доступен в браузере (обычно по адресу:* `http://localhost:5173` *или* `http://localhost:3000`*).*

## 🧠 Машинное обучение и Аналитика

В проекте реализованы модули для продвинутого анализа. Ключевые возможности:
* **Анализ сезонности и трендов.**
* **Умное обнаружение аномалий (Anomaly Detection)** в данных.
* **Матрицы корреляций и проверка гипотез.**

Если необходимо переобучить модели на ваших данных:
```bash
python backend/app/ml/train_model.py
```
